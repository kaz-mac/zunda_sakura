# 会話音声をずんだもん語に変換するデバイス for M5Stack ATOMS3R

## これは何？
喋った言葉をずんだもんの口調「～のだ」変えて、ずんだもんの声に変換するマイクです。
マイクに向かって「今日の天気は晴れです」と喋ると「今日の天気は晴れなのだ」と返ってきます。

さくらインターネットが提供するAIサービス「[さくらのAI Engine](https://www.sakura.ad.jp/aipf/ai-engine/)」を使用して動作します。
